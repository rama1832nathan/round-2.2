Minimum Number of Platforms

You are given arrival and departure times of trains in a railway station. The task is to find the minimum number of platforms required so that no train has to wait.

sample input:

arrivals = [900, 940, 950, 1100, 1500, 1800]
departures = [910, 1200, 1120, 1130, 1900, 2000]

output:

3

current time complexity:o(n2)

current space complexity:o(1)

sample input:

nums1 = [1, 2, 2, 3]
nums2 = [2, 2]
